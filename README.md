# dext-search-plugin

> [Dext](https://github.com/vutran/dext) plugin to search [Google](https://google.com).

## Install

```bash
$ dpm install dext-search-plugin
```

*Requires [Dext](https://github.com/vutran/dext).*

## Usage

In Dext, type `search`, then your query.
